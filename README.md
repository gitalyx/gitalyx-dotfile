### Installation

  ```sh
  git clone https://github.com/gitalyx/gitalyx-dotfile.git
  ```

<h1>On arch linux.</h1>

```sh
cd gitalyx-dotfile
chmod +x arch_install.sh
./arch_install.sh
```

<h1>On nixos.</h1>

```sh
cd gitalyx-dotfile
chmod +x arch_install.sh
bash nix_install.sh
```
